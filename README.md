# ✈️ SAP ALV Airline Analytics Dashboard

## 📌 Overview
This project is a professional ABAP-based interactive dashboard built using ALV Grid. It enables efficient analysis of airline data and simulates real-time reporting used in enterprise SAP systems.

---

## 🎯 Objective
To design a dynamic and user-friendly reporting system that provides insights into airline operations such as pricing, revenue, and flight distribution.

---

## 🚀 Key Features
- 📊 Interactive ALV Grid using `CL_GUI_ALV_GRID`
- 🔍 Dynamic filtering (Airline, Date Range, Price Range)
- 💰 Revenue calculation (Price × Seats Occupied)
- 🎨 Traffic light color coding for better visualization
- 📤 Excel export functionality
- 📈 Analytics popup (Total Flights, Average Price, Total Revenue)
- 🖱️ Double-click interaction (drill-down simulation)
- ⚙️ Modular and clean ABAP architecture

---

## 🧠 Tech Stack
- SAP ABAP
- SAP GUI
- ALV Grid Control (Object-Oriented)

---

## ⭐ Unique Highlights
- Event-driven UI with toolbar customization  
- Real-time analytics computation  
- Clean separation of logic and presentation  
- Industry-style ALV implementation 

---

## ⚠️ Note
This project is developed using standard ABAP syntax and architecture. Due to limited SAP system access, execution is represented through expected outputs.
